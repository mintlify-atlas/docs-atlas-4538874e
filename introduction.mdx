# Welcome to HozonDB

HozonDB is a SQLite-like database built from scratch in Rust. It's a learning project that implements core database concepts including page-based storage, SQL parsing and execution, and performance benchmarking.

<Note>
HozonDB is a learning project built in public. It demonstrates fundamental database internals including storage engines, query execution, and transaction management.
</Note>

## Architecture Overview

HozonDB consists of several core components:

- **Page Manager** (`src/storage/page.rs`): Handles 4KB page allocation, reading, and writing with metadata tracking
- **Table Catalog** (`src/catalog/`): Manages table schemas, rows, and column definitions
- **SQL Parser** (`src/sql/parser.rs`): Converts SQL text into executable statements
- **Executor** (`src/sql/executor.rs`): Executes parsed statements and returns results
- **REPL** (`src/repl.rs`): Interactive command-line interface
- **Benchmark Runner** (`src/benchmark/runner.rs`): Performance testing framework

## Quick Example

Here's a simple example of using HozonDB:

```sql
hozondb> .open test.hdb
Opened database file: test.hdb

hozondb> CREATE TABLE users (id INTEGER, name TEXT);
Table 'users' created.

hozondb> INSERT INTO users VALUES (1, 'Alice');
1 row inserted.

hozondb> SELECT * FROM users;
| id | name |
| 1 | "Alice" |
```
