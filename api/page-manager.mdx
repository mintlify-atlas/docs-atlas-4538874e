The `PageManager` struct handles page-based storage operations, providing the foundation for HozonDB's persistence layer.

## Methods

### new

Opens an existing database or creates a new one.

```rust
pub fn new(path: &str) -> io::Result<Self>
```

<ParamField path="path" type="&str" required>
  Path to the database file (e.g., "mydb.hdb")
</ParamField>

<ResponseField name="return" type="io::Result<PageManager>">
  - `Ok(PageManager)` - Successfully opened/created database
  - `Err` - File I/O error or lock acquisition failure
</ResponseField>

**Behavior:**
- Creates database file if it doesn't exist
- Validates magic number on existing files
- Acquires exclusive lock (`.hdb.lock` file)
- Initializes with 1 page (header page)

**Example:**
```rust
use hozondb::storage::page::PageManager;

// Create new database
let mut pm = PageManager::new("mydb.hdb")?;

// Attempt concurrent access (fails)
let pm2 = PageManager::new("mydb.hdb"); // Err: WouldBlock
```

### allocate_page

Allocates a new page and returns its ID.

```rust
pub fn allocate_page(&mut self) -> io::Result<PageId>
```

<ResponseField name="return" type="io::Result<PageId>">
  ID of the allocated page (1-indexed, page 0 is reserved for headers)
</ResponseField>

**Behavior:**
- Reuses freed pages if available (LIFO order)
- Extends file if no free pages exist
- Initializes page metadata

**Example:**
```rust
let page1 = pm.allocate_page()?; // Returns 1
let page2 = pm.allocate_page()?; // Returns 2
println!("Total pages: {}", pm.num_pages()); // 3 (including header)
```

### free_page

Adds a page to the free list for reuse.

```rust
pub fn free_page(&mut self, page_id: PageId) -> io::Result<()>
```

<ParamField path="page_id" type="PageId" required>
  ID of the page to free
</ParamField>

**Behavior:**
- Adds page to head of free list (LIFO)
- Page can be reused by next `allocate_page()` call
- Does not shrink file size

**Example:**
```rust
let page = pm.allocate_page()?; // 1
pm.free_page(page)?;

let reused = pm.allocate_page()?; // 1 (reused)
assert_eq!(reused, page);
```

## File Structure

### Database Header (Page 0, first 12 bytes)

| Offset | Size | Description |
|--------|------|-------------|
| 0 | 4 | Magic number (0x484F5A4E) |
| 4 | 4 | Total number of pages |
| 8 | 4 | First free page ID (or 0xFFFFFFFF if none) |

### Page Metadata (first 9 bytes of each page)

| Offset | Size | Description |
|--------|------|-------------|
| 0 | 1 | is_full flag (0 or 1) |
| 1 | 2 | last_offset (u16) |
| 3 | 2 | num_rows (u16) |
| 5 | 4 | next_page (u32, 0xFFFFFFFF if none) |

### Page Data (bytes 9-4095)

Actual row data starts at offset 9 and continues to the end of the page.

## Concurrency Control

PageManager uses a lock file to prevent concurrent access:

```rust
// First connection acquires lock
let pm1 = PageManager::new("db.hdb")?; // OK

// Second connection fails
let pm2 = PageManager::new("db.hdb"); // Err: WouldBlock

// Lock released when dropped
drop(pm1);
let pm3 = PageManager::new("db.hdb"); // OK now
```

Lock file: `{database_path}.lock` (e.g., `db.hdb.lock`)

## Free Page Management

Freed pages form a linked list:

```rust
// Allocate 3 pages
let p1 = pm.allocate_page()?; // 1
let p2 = pm.allocate_page()?; // 2
let p3 = pm.allocate_page()?; // 3

// Free in order: p1, p2, p3
pm.free_page(p1)?;
pm.free_page(p2)?;
pm.free_page(p3)?;

// Free list: 3 → 2 → 1 → NULL (LIFO)

// Reallocate in LIFO order
let r1 = pm.allocate_page()?; // 3
let r2 = pm.allocate_page()?; // 2
let r3 = pm.allocate_page()?; // 1
```

## Example: Multi-Page Table

```rust
use hozondb::storage::page::{PageManager, PageMetadata, PAGE_DATA_START, PAGE_SIZE};

let mut pm = PageManager::new("table.hdb")?;

// Allocate first page for table
let page1 = pm.allocate_page()?;
let mut page1_data = [0u8; PAGE_SIZE];

// Write some rows...
let row_data = b"row1,row2,row3";
page1_data[PAGE_DATA_START..PAGE_DATA_START + row_data.len()]
    .copy_from_slice(row_data);

// Page is full, need another page
let page2 = pm.allocate_page()?;

// Link pages
let meta = PageMetadata {
    is_full: true,
    last_offset: PAGE_SIZE,
    num_rows: 3,
    next_page: Some(page2),
};

PageManager::update_metadata_in_buffer(&mut page1_data, &meta);
pm.write_page(page1, &page1_data)?;

// Traverse page chain
let mut current = page1;
loop {
    let meta = pm.read_page_metadata(current)?;
    println!("Page {} has {} rows", current, meta.num_rows);
    
    match meta.next_page {
        Some(next) => current = next,
        None => break,
    }
}
```
